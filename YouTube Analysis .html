Exploring YouTube Channel Performance: A Comprehensive Data Analysis

This project is a descriptive and exploratory data analysis (EDA), focusing on uncovering trends, correlations, and patterns within YouTube channel data.

Objective:

The primary objective of this analysis was to explore relationships between various metrics and to identify patterns that contribute to the success of YouTube channels.

Data Source:

https://www.kaggle.com/datasets/nelgiriyewithana/global-youtube-statistics-2023

Data Cleaning:

The dataset required significant preprocessing to handle missing values, non-standard date formats, and non-numeric characters in key columns like subscriber counts. This step was crucial in ensuring the accuracy and reliability of the subsequent analysis.

Key Research Questions:

1.What are the key factors affecting the success of a YouTube channel in terms of views and subscriber growth? Is there a relationship between the number of subscribers and video views, and how significant is this correlation?

2.Which content categories generally achieve better performance? How do different metrics, such as content categories and location, influence a video's overall success?

3.What patterns or trends can be observed in the geographic distribution of viewers and subscribers?

4.What are the top ten channels with the highest number of subscribers and views?

5.How has YouTube channel growth evolved over the years?

# Fill missing values in 'created_date' with default value (1)
df_cleaned['created_date'].fillna(1, inplace=True)

# Create a datetime column using valid year, month, and date
def create_datetime(year, month, day):
    try:
        # Return datetime object
        return datetime(year=int(year), month=int(month), day=int(day))
    except (ValueError, TypeError):
        # Return Not a Time if there's an error
        return pd.NaT

# Apply the function to create a datetime column
df_cleaned['created_date'] = df_cleaned.apply(
    lambda row: create_datetime(row['created_year'], row['created_month'], row['created_date']),
    axis=1
)

# Extract year for grouping, only if created_date is not NaT
df_cleaned['year'] = df_cleaned['created_date'].dt.year

# Group by the year column and count the number of entries
channel_growth = df_cleaned.groupby('year').size()

# Plotting the channel growth over time
plt.figure(figsize=(10, 6))
channel_growth.plot(kind='line', marker='o', color='b')
plt.title('YouTube Channel Growth Over Time')
plt.xlabel('Year')
plt.ylabel('Number of Channels Created')
plt.grid(True)
plt.show()

This line graph tracks the number of YouTube channels created over time. There's a significant growth in the number of channels post-2005, coinciding with YouTube's rise in popularity. The peak around 2010-2015 suggests a period of rapid platform adoption, followed by a decline, possibly indicating market saturation or changes in platform policies.
